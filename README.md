## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/AutoTestUploadProjectAi/automated-deployment-coordinator.git
cd automated-deployment-coordinator
```

### 2. Install Dependencies

```bash
dotnet restore
```

### 3. Run the Application

```bash
dotnet run
```
